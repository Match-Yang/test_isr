# API Overview

---

ZEGOCLOUD AI Agent server provides the following relevant API interfaces and callbacks. You can refer to [Accessing Server APIs](./accessing-server-apis.mdx) to call the interfaces below.

## Agent Configuration Management

| Interface Name                                                            | Interface Function           | Default Call Frequency Limit |
| ------------------------------------------------------------------- | ------------------ | ---------------- |
| [Register An Agent](./agent-configuration-management/register-agent.mdx)                | Register an agent (Agent) for creating an agent instance.       | 10 times/second         |
| [Unregister An Agent](./agent-configuration-management/unregister-agent.mdx)                | Unregister an agent.           | 10 times/second         |
| [Update An Agent](./agent-configuration-management/update-agent.mdx)                | Modify an agent.          | 10 times/second         |
| [List Agents](./agent-configuration-management/list-agents.mdx)     | Get the list of available agents.       | 10 times/second         |
| [Query Agent Details](./agent-configuration-management/query-agents.mdx)     | Get the details of a specified agent.       | 10 times/second         |

## Agent Instance Management

| Interface Name                                                            | Interface Function           | Default QPS limit |
| ------------------------------------------------------------------- | ------------------ | ---------------- |
| [Create An Agent Instance](./agent-instance-management/create-agent-instance.mdx) | Create an agent instance and add it to text conversations and voice (RTC) conversations. | 10 times/second         |
| [Update An Agent Instance](./agent-instance-management/update-agent-instance.mdx)       | Update an existing agent instance. | 10 times/second         |
| [Delete An Agent Instance](./agent-instance-management/delete-agent-instance.mdx)       | Delete an existing agent instance. | 10 times/second         |

## Agent Instance Control

| Interface Name                                                            | Interface Function           | Default QPS limit |
| ------------------------------------------------------------------- | ------------------ | ---------------- |
| [Trigger LLM](./agent-instance-control/send-agent-instance-llm.mdx)               | Trigger the LLM service as a user, and based on the response content from the LLM, proactively invoke the TTS service as an agent to send voice messages to the user.      | 10 times/second         |
| [Trigger TTS](./agent-instance-control/send-agent-instance-tts.mdx)               | Trigger the TTS service as an agent to send voice messages to the user.     | 10 times/second         |
| [Query The Status of An AI Agent Instance](./agent-instance-control/query-agent-instance-status.mdx)               | Query the runtime status information of the agent during voice conversations.     | 10 times/second         |
| [Interrupt Agent Instance](./agent-instance-control/interrupt-agent-instance.mdx)               | Interrupt the agent instance in a voice conversation    | 10 times/second         |
| [Get Agent Instance Context](./agent-instance-control/get-agent-instance-msg-list.mdx)               | get the message list as context used by the agent instance during a voice conversation.    | 10 times/second         |
| [Reset Agent Instance Context](./agent-instance-control/reset-agent-instance-msg-list.mdx)               | Reset the message list as context used by the agent instance during a voice conversation.    | 10 times/second         |


## Callbacks

| Callback Name | Callback Description |
| ------------- | -------------------- |
| [Receiving Callback](./../callbacks/receiving-callback.mdx) |   With this callback, listen to events that occur during the user's conversation with the agent.       |
