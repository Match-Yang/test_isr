# Unregister An AI Agent

---

## Description

By passing the agent ID to this interface, you can deregister an agent.

<PostPrototype link="https://aigc-aiagent-api.zegotech.cn?Action=UnregisterAgent" />

## Request Parameters

Listed below are the parameters specific to this request and part of the common request parameters. For the complete list of common request parameters, see [Accessing Server APIs - Common request parameters](./../accessing-server-apis.mdx#common-request-parameters).

| Parameter          | Type    | Required | Description                                                                 |
|--------------------|---------|----------|-----------------------------------------------------------------------------|
| AgentId           | String  | Yes      | The unique identifier of a registered agent.                                  |

## Sample Request

- Request URL:

    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=UnregisterAgent
    &<Common Request Parameters>
    ```

- Request body:
    ```json
    {
        "AgentId": "zy-123"
    }
    ```

## Response Parameters

| Parameter | Type   | Description                                                                 |
|-----------|--------|-----------------------------------------------------------------------------|
| Code      | Number | Return code. 0 indicates success, other values indicate failure. For more information about error codes and response handling recommendations, please refer to [Return Codes](./../return-codes.mdx). |
| Message   | String | Explanation of the request result.                                           |
| RequestId | String | Request ID.                                                                 |
