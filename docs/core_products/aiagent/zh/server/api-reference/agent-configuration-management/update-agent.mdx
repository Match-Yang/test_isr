# 修改智能体

---

##  描述

通过向本接口传入智能体 ID 和基础配置，您可以更新一个现有的智能体。

import PostPrototype from '/snippets/common/zh/server/post-prototype.mdx'

<PostPrototype link="https://aigc-aiagent-api.zegotech.cn?Action=UpdateAgent" />

## 请求参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表请参考 [调用方式 - 公共请求参数](./../accessing-server-apis.mdx#公共请求参数)。

| 参数               | 类型   | 是否必选 | 描述             |
|------------------|------|------|-------------------------|
| AgentId          | String | 是    | 已注册智能体的唯一标识。                                                             |
| Name            | String | 否       | 智能体名称，长度不超过 64 字节。                                                                 |
| LLM             | Object | 否       | 大语言模型参数。本参数结构，请见 [智能体参数说明 - LLM](./../common-parameter-description.mdx#llm)。                                                            |
| TTS             | Object | 否       | 文本转语音参数。本参数结构，请见 [智能体参数说明 - TTS](./../common-parameter-description.mdx#tts)。                                                              |
| ASR             | Object | 否       | 语音识别参数。本参数结构，请见 [智能体参数说明 - ASR](./../common-parameter-description.mdx#asr)。                                                              |

<Note title="说明">
只有传入本接口的参数才会生效，未传入的参数不会更新。
</Note>
## 请求示例

- 请求地址 URL：

    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=UpdateAgent
    &<公共请求参数>
    ```

- 请求消息体：
    ```json
    {
        "AgentId": "xiaozhi",
        "LLM": {
            "Url": "https://api.minimax.chat/v2/text/chatcompletion_v2",
            "ApiKey": "zego_test", // your api key (zego_test 在接入测试期间(AI Agent 服务开通 2 周内)可使用，详情请查看快速开始说明)
            "Model": "abab6.5s-chat",
            "SystemPrompt": "你是小智，成年女性，是**即构科技创造的陪伴助手**，上知天文下知地理，聪明睿智、热情友善。\n对话要求：1、按照人设要求与用户对话。\n2、不能超过100字。"
        },
        "TTS": {
            "Vendor": "ByteDance",
            "Params": {
                "app": {
                    "appid": "zego_test", // your appid (zego_test 在接入测试期间(AI Agent 服务开通 2 周内)可使用，详情请查看快速开始说明)
                    "token": "zego_test", // your token (zego_test 在接入测试期间(AI Agent 服务开通 2 周内)可使用，详情请查看快速开始说明)
                    "cluster": "volcano_tts"
                },
                "audio": {
                    "voice_type": "zh_female_wanwanxiaohe_moon_bigtts" // your voice type
                }
            }
        },
        "ASR": {}
    }
    ```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Number | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../return-codes.mdx)。 |
| Message | String | 请求结果的说明信息。 |
| RequestId | String | 请求 ID。 |
