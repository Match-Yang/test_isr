# Release Notes

## Version 3.14.0

**Release Date: 2025-05-29**

<h5>New Features</h5>

| <div style={{width:"130px"}}>Feature</div> |Description | <div style={{width:"200px"}}>Related APIs</div> |
| :------------------------------------------ | :----------- | :------------ |
| iOS PIP | Live Streaming Kit supports iOS PIP by default. To disable the feature, you can set `ZegoLiveStreamingPIPIOSConfig.pip.iOS.support` to `false`. | [ZegoLiveStreamingPIPIOSConfig.pip.iOS.support](https://pub.dev/documentation/zego_uikit_prebuilt_live_streaming/latest/zego_uikit_prebuilt_live_streaming/ZegoLiveStreamingPIPIOSConfig/support.html) |
| Rich Media Playback | Supports direct play of rich media (video/audio) by setting `support` of `mediaPlayer.defaultPlayer` in `ZegoUIKitPrebuiltLiveStreamingConfig` to `true`. | <ul><li>[ZegoLiveStreamingMediaPlayerConfig](https://pub.dev/documentation/zego_uikit_prebuilt_live_streaming/latest/zego_uikit_prebuilt_live_streaming/ZegoLiveStreamingMediaPlayerConfig-class.html)</li><li>[ZegoLiveStreamingMediaPlayerDefaultPlayerConfig](https://pub.dev/documentation/zego_uikit_prebuilt_live_streaming/latest/zego_uikit_prebuilt_live_streaming/ZegoLiveStreamingMediaPlayerDefaultPlayerConfig-class.html)</li></ul> |
| Screen Sharing Button | If you want to add the screen sharing button to the top toolbar, which is not displayed by default, you can configure it via `topMenuBar.buttons`. | <ul><li>[ZegoLiveStreamingTopMenuBarConfig > buttons](https://pub.dev/documentation/zego_uikit_prebuilt_live_streaming/latest/zego_uikit_prebuilt_live_streaming/ZegoLiveStreamingTopMenuBarConfig/buttons.html)</li><li>[ZegoLiveStreamingMenuBarButtonName > toggleScreenSharingButton](https://pub.dev/documentation/zego_uikit_prebuilt_live_streaming/latest/zego_uikit_prebuilt_live_streaming/ZegoLiveStreamingMenuBarButtonName.html#toggleScreenSharingButton)</li></ul> |
| Auto Start | The media player supports automatically starting playback when a media URL is set. | [ZegoUIKitMediaPlayerConfig > autoStart](https://pub.dev/documentation/zego_uikit_prebuilt_live_streaming/latest/zego_uikit_prebuilt_live_streaming/ZegoUIKitMediaPlayerConfig/autoStart.html) |
| Microphone State Icon | Supports hiding the microphone icon in the audioVideoView. | [ZegoLiveStreamingAudioVideoViewConfig > showMicrophoneStateOnView](https://pub.dev/documentation/zego_uikit_prebuilt_live_streaming/latest/zego_uikit_prebuilt_live_streaming/ZegoLiveStreamingAudioVideoViewConfig/showMicrophoneStateOnView.html) |

<h5>Enhancements</h5>

- Improved the internal minimized interface within CallKit.
