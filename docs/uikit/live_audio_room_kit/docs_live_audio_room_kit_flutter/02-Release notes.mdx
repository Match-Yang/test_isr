# Release Notes

## Version 3.16.0

**Release Date: 2025-05-29**

<h5>New Features</h5>

| <div style={{width:"130px"}}>Feature</div> |Description | <div style={{width:"200px"}}>Related APIs</div> |
| :------------------------------------------ | :----------- | :------------ |
| Rich Media Playback | Supports direct play of rich media (video/audio) by setting `support` of `mediaPlayer.defaultPlayer` in `ZegoLiveAudioRoomMediaPlayerConfig` to `true`. | <ul><li>[ZegoLiveStreamingMediaPlayerConfig](https://pub.dev/documentation/zego_uikit_prebuilt_live_audio_room/latest/zego_uikit_prebuilt_live_audio_room/ZegoLiveAudioRoomMediaPlayerConfig-class.html)</li><li>[ZegoLiveStreamingMediaPlayerDefaultPlayerConfig](https://pub.dev/documentation/zego_uikit_prebuilt_live_audio_room/latest/zego_uikit_prebuilt_live_audio_room/ZegoLiveAudioRoomMediaPlayerDefaultPlayerConfig-class.html)</li></ul> |
| Member List Button | If you want to add the member list button to the top toolbar, which is not displayed by default, you can configure it via `topMenuBar.buttons`. | <ul><li>[ZegoLiveAudioRoomTopMenuBarConfig > buttons](https://pub.dev/documentation/zego_uikit_prebuilt_live_audio_room/latest/zego_uikit_prebuilt_live_audio_room/ZegoLiveAudioRoomTopMenuBarConfig/buttons.html)</li><li>[ZegoLiveAudioRoomMenuBarButtonName > showMemberListButton](https://pub.dev/documentation/zego_uikit_prebuilt_live_audio_room/latest/zego_uikit_prebuilt_live_audio_room/ZegoLiveAudioRoomMenuBarButtonName.html#showMemberListButton)</li></ul> |
| Customizable Seat Area | Support fully customizable seat area, using `topLeft` positioning, specifying the size via `containerSize`, and customizing the seat display via `containerBuilder`. | <ul><li>[ZegoUIKitPrebuiltLiveAudioRoomConfig > ZegoLiveAudioRoomSeatConfig](https://pub.dev/documentation/zego_uikit_prebuilt_live_audio_room/latest/zego_uikit_prebuilt_live_audio_room/ZegoLiveAudioRoomSeatConfig-class.html)</li><li>[ZegoLiveAudioRoomSeatConfig > topLeft](https://pub.dev/documentation/zego_uikit_prebuilt_live_audio_room/latest/zego_uikit_prebuilt_live_audio_room/ZegoLiveAudioRoomSeatConfig/topLeft.html)</li><li>[ZegoLiveAudioRoomSeatConfig > containerSize](https://pub.dev/documentation/zego_uikit_prebuilt_live_audio_room/latest/zego_uikit_prebuilt_live_audio_room/ZegoLiveAudioRoomSeatConfig/containerSize.html)</li><li>[ZegoLiveAudioRoomSeatConfig > containerBuilder](https://pub.dev/documentation/zego_uikit_prebuilt_live_audio_room/latest/zego_uikit_prebuilt_live_audio_room/ZegoLiveAudioRoomSeatConfig/containerBuilder.html)</li></ul> |
