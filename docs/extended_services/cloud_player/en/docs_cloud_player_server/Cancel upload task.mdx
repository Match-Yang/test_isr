# Cancel an upload task

- - -

## Overview

Call this method to cancel an upload task.

<Note title="Note">

- This method can only cancel tasks that are pending; it is not applicable to tasks that have started or completed.
- To check the cancellation result, please use the [callback](./Callback/Callback.mdx) or the [Describe upload tasks](./Describe%20Upload%20Tasks.mdx) method.
</Note>

## Request method and endpoint

- Request method: POST
- Request URL: `https://cloud-player-api.zego.im/?Action=CancelUploadTask`
- Protocol: HTTPS
- Rate limit: 20 times/second

## Request parameters

The following list only lists the interface request parameters. For common parameters, please refer to [Accessing server APIs - Common request parameters](./Accessing%20Server%20APIs.mdx#common-parameters).

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| UploadTaskId | String | Yes | The unique identifier for the upload task. After calling the [CreateUploadTask](./Create%20Upload%20task.mdx) API to create an upload task, you can obtain this ID through the response. |

## Request example

- Request URL
    ```bash
    https://cloud-player-api.zego.im/?Action=CancelUploadTask
    &AppId=1234567890
    &SignatureNonce=15215528852396
    &Timestamp=1234567890
    &Signature=7a2c0f11145fb760d607a07b54825013
    &SignatureVersion=2.0
    &IsTest=false
    ```

- Request body
    ```json
    {
        "UploadTaskId": "XXXXXXXXXXXX"
    }
    ```

## Response parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| Code | Number | Error code. |
| Message | String | Error description. |
| RequestId | String | Request ID. |
