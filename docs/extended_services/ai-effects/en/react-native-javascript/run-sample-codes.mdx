## Prerequisites

Please ensure your development environment meets the following requirements:

* React Native 0.73.6 or above.
* iOS 13.0 or above and an iOS device or simulator that supports audio and video (real device recommended).
* Android 8.0 or above and an Android device or simulator that supports audio and video (real device recommended). For real devices, please enable "USB debugging".
* iOS/Android device connected to the Internet.
* [Node.js](https://nodejs.org/en/) installed, LTS version recommended as shown on their homepage.

## Requirements

Please ensure the following requirements are met before running:
* Registered an account on [ZEGOCLOUD Console](https://console.zegocloud.com/account/login) and obtained the AppID and AppSign required for SDK initialization.
* Contacted ZEGOCLOUD technical support to obtain the License file for beauty effects.

## Run the Sample Code

1. Download the latest version of [iOS ZegoEffects SDK](/ai-effects-ios-objc/downloads) or [Android ZegoEffects SDK](/ai-effects-android-java/downloads), extract it, and import the `Resources` and `Models` folders into your downloaded [sample code](https://github.com/ZEGOCLOUD/zego_effects_reactnative_demo).

    <Tabs>
    <Tab title="iOS">

    For iOS, manually create an `Assets` folder in the sample code, and place the `Resources` and `Models` folders inside the `Assets` folder. For example, if your sample project is named `example` and you place all resources in the `Assets` folder, after importing, your project structure should look like this:

    ```bash
    ios
    ├── example
    │   ├── AppDelegate.h
    │   ├── AppDelegate.mm
    │   ├── Images.xcassets
    ├── Assets
    │   ├── Models
    │   │   ├── FaceDetectionModel.model
    │   │   └── SegmentationModel.model
    │   └── Resources
    │       ├── ColorfulStyleResources
    │       ├── CommonResources.bundle
    │       ├── FaceWhiteningResources.bundle
    │       ├── MakeupResources
    │       ├── PendantResources.bundle
    │       ├── RosyResources.bundle
    │       └── TeethWhiteningResources.bundle
    │
    ```
    </Tab>
    <Tab title="Android">
    For Android, add the `Resources` and `Models` folders to the `assets` directory in your sample code. Typically, resources should be placed in the `android/app/src/main/assets` directory, so after importing, your sample code project structure should look like this:

    ```bash
    android/app/src/main
    ├── AndroidManifest.xml
    ├── assets
    │   ├── Models
    │   │   ├── FaceDetectionModel.model
    │   │   └── SegmentationModel.model
    │   └── Resources
    │       ├── ColorfulStyleResources
    │       ├── CommonResources.bundle
    │       ├── FaceWhiteningResources.bundle
    │       ├── MakeupResources
    │       ├── PendantResources.bundle
    │       ├── RosyResources.bundle
    │       └── TeethWhiteningResources.bundle
    ├── java
    └── res
    ```
    </Tab>
    </Tabs>

2. Add a `KeyCenter.js` file in the root directory of the sample code with the following content. Fill in your `appID`, `appSign`, and `effectLicense`. For how to obtain these, please refer to [Prerequisites](#prerequisites).

    ```js
    const KeyCenter = { 
        // Get appID from https://console.zegocloud.com/account/login
        // For example: const appID = 123456789;
        appID: ,
        // Get appSign from https://console.zegocloud.com/account/login, for example: "39011cbxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
        appSign: '',
        effectLicense: '',
    }

    export default KeyCenter
    ```

4. In the root directory, install dependencies and run the sample code.

    ```bash
    yarn

    # Run iOS
    yarn ios

    # Run Android
    yarn android
    ```
